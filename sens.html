Les Sens


Collaborator(s): n/a

Les Sens is an interactive digital publication for a tablet. The viewer is presented with displays that interact with the five senses in order to aid in learning a language. They are able to learn names of everyday, basic objects but also how to say them. At the end of the publication, there is a mini quiz to assess knowledge retained through the experience.

The language for the display is French.
